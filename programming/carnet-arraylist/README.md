# IntBasAp - Activité 05-01 - carnet, gestion d'adresses - tableau dynamique

Ce projet apporte des  élements de solution pour l'activité
[IntBasAp - Activité 05-01 - Gestion d'un carnet d'adresses](https://mylos.cifom.ch/cours/int-prog1-ap1/activites/basap-activite-05-01-carnet/).

Cette implémentation utilise une tableau qui grandit à chaque ajout d'un nouveau contact et retrécit à la suppression d'un contact.

* L'analyse est la même que le projet [carnet-tableau](https://mylos.cifom.ch/gitlab/dhu.cours/projets/carnet/carnet-tableau/).
